# FEPI - Calculadora de Médias

Calculadora automatizada de médias para o Centro Universitário de Itajubá (FEPI).

## Descrição
Projeto web desenvolvido para ajudar os alunos e o corpo docente a calcular facilmente a média semestral e bimestral, as notas necessárias para o exame final (recuperação) e verificar o status de aprovação.

## Funcionalidades
- Cálculo de médias semestrais e bimestrais.
- Verificação de status de aprovação.
- Projeção de notas necessárias para o exame final.

## Tecnologias Utilizadas
- HTML5
- CSS3 (Variáveis CSS, Flexbox, Grid)
- JavaScript (Vanilla)
